AdventureScape

By Savannah Stumpf

Application Overview

AdventureScape will be a text-based adventure game. The user will choose how they want the story to go by entering their input. Different inputs make the story take different paths.

Technology Overview

AdventureScape will be written with C++ in Visual Studio. It will be compatible with Windows and macOS.

Proposal last updated: 11/4/2022

Feature List

  • Interactive story
  • Diverging story paths
  • Multiple endings

Future Feature List

  • Background music
  • Pictures depicting the specific setting the player is in

Basic Technical Features

  • Loops
  • Variables
  • Strings

Advanced Technical Features
